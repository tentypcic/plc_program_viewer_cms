Eksport bloków z TIA Portal do PDF

W TIA Portal kliknij prawym przyciskiem na wybrany blok w drzewie projektu i wybierz Print…. W oknie wydruku ustaw drukarkę na Microsoft Print to PDF, po wybraniu ustawień pliku kliknij Print, a następnie zapisz plik PDF. Nazwa pliku powinna odpowiadać nazwie bloku (np. Main_OB1.pdf).

Dodawanie bloku do drzewka projektu

W menu mamy podstawowe 3 przyciski, kliknij Edit Mode aby rozwinąć menu, następnie wybierz Add Block, uzupełnij Block Name, wskaż ścieżkę do PDF (np. pdf/main.pdf), a następnie zatwierdź. Opcjonalnie możesz zaznaczyć Create F-Block (yellow icon) dla bloków bezpieczeństwa.

Exporting blocks from TIA Portal to PDF

In TIA Portal, right-click on the selected block in the project tree and choose Print…. In the print window set the printer to Microsoft Print to PDF. After selecting the file settings, click Print, and then save the PDF file. The file name should match the block name (e.g. Main_OB1.pdf).

Adding a block to the project tree

In the menu we have 3 basic buttons. Click Edit Mode to expand the menu, then select Add Block, fill in the Block Name, provide the path to the PDF (e.g. pdf/main.pdf), and confirm. Optionally, you can check Create F-Block (yellow icon) for safety blocks.
